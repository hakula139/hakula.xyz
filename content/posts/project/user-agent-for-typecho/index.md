## 使用说明

1. 解压并修改文件夹名为 `UserAgent`，将插件上传至网站目录的 `/usr/plugins` 下。
2. 在 Typecho 后台「插件管理」处启用插件。
3. 在需要显示 UserAgent 的地方插入以下代码[^1]。

```php
<?php UserAgent_Plugin::render($comments->agent); ?>
```
